Each pixel's luminance is computed as L = 0.299R + 0.587G + 0.114B. Pixels below the threshold (and with > 10 alpha) are marked inside (dark shapes); Invert flips this for light shapes. The alpha array is passed to maskRegion(w, h, alpha, threshold), which turns scan-line runs into horizontal spans. shapeFlow() pours text across those spans row by row. Drag any photo into the canvas to use your own silhouette — nothing is uploaded.

raw Pretext, by hand
// the whole demo is the geometry. spansAt(y):
function spansAt(yc) {
  const row = Math.floor(yc), spans = [];
  let inRun = false, start = 0;
  for (let x = 0; x < W; x++) {
    const inside = insideMask[row*W + x] >= 128; // luminance threshold
    if (inside && !inRun) { inRun = true; start = x; }
    else if (!inside && inRun) { inRun = false; spans.push([start, x]); }
  }
  if (inRun) spans.push([start, W]);
  return spans;
}

The whole demo is the geometry: luminance threshold + per-row run-length = maskRegion. The text loop is the trivial part.
